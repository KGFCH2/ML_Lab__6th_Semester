## 📚 Lab Sessions (Day-wise)

Explore the detailed implementation and code snippets for each session via the links below:

### 📉 [Day 1: Simple Linear Regression](./Day_1/README.md)

*Focuses on implementing the regression model from scratch using mathematical cross-deviation.*

* 🔢 **Logic:** Calculation of $\beta_0$ and $\beta_1$ using SSxy/SSxx.
* 📊 **Results:** Visualizing data points vs the regression line.
* 🧠 **Skill:** Manual implementation without Scikit-Learn.

### 📉 [Day 2: Gradient Descent Optimization](./Day_2/README.md)

*Focuses on iterative weight optimization and feature normalization.*

* ⚙️ **Logic:** Implementation of the Gradient Descent algorithm.
* 📏 **Transform:** Feature scaling and normalization techniques.
* 📉 **Analysis:** Monitoring cost function minimization across iterations.

### 📉 [Day 3: Regularization (Lasso & Ridge)](./Day_3/README.md)

*Implementation of L1 and L2 regularization for feature selection and overcoming overfitting.*

* 🎯 **Lasso:** L1 regularization for automated feature selection.
* 🛡️ **Ridge:** L2 regularization for handling multicollinearity.
* 🧪 **Metrics:** Model evaluation using MSE and R² metrics.

### 📉 [Day 4: K-Nearest Neighbors (KNN)](./Day_4/README.md)

*Manual implementation of the KNN classifier using Euclidean distance and majority voting.*

* 📐 **Distance:** Custom Euclidean distance calculation function.
* 🗳️ **Voting:** Majority voting logic for classification.
* 📉 **Plot:** Accuracy analysis vs K-value visualization.

### 📉 [Day 5: Logistic Regression](./Day_5/README.md)

*Implementation of Logistic Regression for binary classification using Salary Data.*

* 🔢 **Logic:** Sigmoid function and gradient descent for parameter optimization.
* 📊 **Data:** Salary dataset for predicting binary outcomes.
* 🧠 **Skill:** Manual implementation of logistic regression algorithm.

### 📊 [Day 6: Clustering Algorithms](./Day_6/README.md)

*Unsupervised learning with K-Means and DBSCAN clustering on Titanic dataset.*

* 🎯 **K-Means:** Elbow method for optimal cluster selection.
* 🔍 **DBSCAN:** Density-based clustering with noise detection.
* 📊 **Visualization:** PCA-reduced cluster plots and elbow curves.
