# Day 8: Ensemble Learning

This folder contains implementations of AdaBoost and Random Forest using the `titanic_toy.csv` dataset.

## Files

- [AdaBoost.py](AdaBoost.py): Implementation of AdaBoost Classifier.
- [RandomForest.py](RandomForest.py): Implementation of Random Forest Classifier.
- [titanic_toy.csv](titanic_toy.csv): The dataset used for training and testing.

## How to run

1. Ensure you have the required libraries installed:

   ```bash
   pip install pandas scikit-learn
   ```

2. Run the scripts:

   ```bash
   python AdaBoost.py
   python RandomForest.py
   ```
